# Telegram Bot

Простой Telegram-бот на Node.js.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Создайте `.env` файл с `BOT_TOKEN=ваш_токен`.
4. Запустите: `npm start` или `npm run dev` для разработки.

## Команды

- `/start` - Приветствие
- `/help` - Список команд
- `/echo <текст>` - Повторить текст

## Разработка

- Линтер: `npm run lint`
- Тесты: `npm test`
